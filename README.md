## 📖 Overview  

This repository contains **lab tasks, assignments, and practice work** completed during the **Database Systems (DBMS)** course.

It focuses on building a strong understanding of **database design, SQL queries, and relational database concepts**, along with practical implementations.

---

## 📂 Repository Contents  

---

### 🧪 Lab Tasks  

- Weekly lab exercises covering SQL and database concepts  
- Hands-on practice with queries and database operations  
- Step-by-step implementation of relational databases  

---

### 📝 Assignments  

- Structured coursework problems based on DBMS concepts  
- Focus on designing and querying databases  
- Includes complex query-solving and optimization  

---

## 🧠 Topics Covered  

---

### 🗂️ Database Design  
- Entity-Relationship (ER) Diagrams  
- Schema Design  
- Relationships & Constraints  

---

### 🧮 SQL Fundamentals  
- Creating Tables (`CREATE TABLE`)  
- Modifying Tables (`ALTER TABLE`)  
- Inserting, Updating, Deleting Data  

---

### 🔗 Advanced SQL Queries  
- Joins (Inner, Left, Right, Full)  
- Subqueries  
- Nested Queries  
- Aggregation Functions  
- Group By & Having Clauses  

---

### ⚙️ Advanced Concepts  
- Triggers  
- Views  
- Stored Procedures (if included)  
- Indexing Basics  

---

### 📚 Normalization  
- 1NF, 2NF, 3NF, BCNF  
- Removing redundancy  
- Functional Dependencies  

---

## ▶️ How to Run  

### 💻 Using SQL Environment  

1. Install any SQL tool:
   - MySQL / SQL Server / Oracle / PostgreSQL  
2. Open your database environment  
3. Run `.sql` files from this repository  
4. Execute queries and observe results
